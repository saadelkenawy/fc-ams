## Section 3 — Digital Receipt Logging System

> This specification is implemented as the **Procurement Module** within the Fadl Clinic Management System (see `/services/procurement-service` and `/frontend/web-portal/src/app/(dashboard)/procurement`). The tables below serve as the authoritative developer reference.

### 3a — Field Definitions

| Field name | Data type | Mandatory | Validation rule | Auto-population source |
|-----------|-----------|-----------|----------------|----------------------|
| item_id | UUID FK → procurement_items | YES (NOT NULL) | Must exist in item catalog | Dropdown from catalog |
| unit_price_egp | DECIMAL(10,2) | YES | > 0; if foreign currency, convert at CBE rate on receipt date; set `currency_source = 'converted'` | Manual entry |
| quantity_received | INTEGER | YES | > 0; if linked PO exists and `|received − ordered| / ordered > 0`, raise DISCREPANCY_ALERT | Manual entry |
| vendor_id | UUID FK → procurement_vendors | YES | Must exist in approved vendor directory | Dropdown |
| invoice_file_uri | TEXT (S3 URI or BLOB path) | YES | Accept PDF or JPEG only; max 10 MB; triggers OCR pipeline on upload | File upload |
| invoice_number | VARCHAR(100) | YES | Auto-populated from OCR; editable if `ocr_confidence < 0.85`; edit flagged in audit trail | OCR extraction |
| invoice_date | DATE | YES | Auto-populated from OCR; must not be a future date | OCR extraction |
| invoice_total_egp | DECIMAL(12,2) | YES | Auto-populated from OCR; flagged if ≠ SUM(unit_price × quantity) ± 1% | OCR extraction |
| batch_lot_number | VARCHAR(100) | Conditional | Required for all EDA-registered items; optional for `eda_status = 'Not regulated'` | Manual entry |
| expiry_date | DATE | Conditional | Required if item has EDA registration or is consumable; EXPIRY_ALERT at T−90 and T−30 days | Manual entry |
| date_received | DATE | YES | Defaults to today(); must not be a future date | System timestamp |
| received_by_staff_id | UUID FK → staff (identity-service) | YES | Must match authenticated session user JWT sub claim | Session auth |
| reorder_alert_threshold | INTEGER | NO | If `current_stock ≤ threshold`, trigger REORDER_ALERT linked to preferred vendor | Catalog default |
| currency_source | ENUM('EGP', 'converted') | YES | 'converted' when original invoice is USD/EUR; requires `cbe_rate` to be set | Derived |
| cbe_rate | DECIMAL(10,4) | Conditional | Required when `currency_source = 'converted'`; stored in `currency_audit_log` JSONB | Manual or API |

### 3b — Workflow Logic

1. **On invoice upload →** OCR pipeline extracts `invoice_number`, `invoice_date`, and `invoice_total_egp` and pre-populates fields; any field modified post-OCR is flagged with `ocr_overridden = true` in the audit trail.
2. **On quantity_received entry →** system compares against linked purchase order `quantity_ordered`; if discrepancy > 0%, raise DISCREPANCY_ALERT to procurement manager via email notification (if SMTP configured) and in-app alert.
3. **On expiry_date entry →** schedule EXPIRY_ALERT jobs at T−90 days (severity: `warning`) and T−30 days (severity: `critical`); alerts sent to clinic administrator via notification service.
4. **On stock update →** if `current_stock ≤ reorder_alert_threshold`, trigger REORDER_ALERT linked to the item's preferred vendor from the directory.
5. **All currency conversions →** log CBE exchange rate and conversion timestamp in `currency_audit_log` JSONB field; this field is immutable after creation.

## Closing Summary — Top 3 Procurement Risks in the Egyptian Clinic Context

**1. Foreign-currency supply chain volatility.** The majority of Class II and III medical devices and consumables (IV cannulas, sutures, diagnostic equipment) are imported and invoiced in USD or EUR. With EGP exchange rate fluctuations exceeding 30% year-on-year in 2023–2024, unit costs in EGP can shift materially between purchase order and delivery. Clinics should: (a) build a 60-day buffer stock for high-turnover consumables; (b) negotiate EGP-denominated contracts where possible with local distributors; (c) log the CBE rate on every foreign-currency receipt (enforced by the digital receipt logging system in Section 3).

**2. EDA registration gaps for specialty and emerging items.** Mesotherapy micro-needles, cryotherapy units, and laser protective eyewear sit in a regulatory grey zone — they may be imported under general import permits rather than full EDA device registration. Procuring unregistered or un-permitted items exposes the clinic to inspection liability under Law No. 127 of 1955 and its medical device annexes. Clinics must request the EDA registration certificate or valid import permit from the supplier before accepting delivery of any Class II or III item.

**3. Limited local manufacturing depth.** While Egypt has a growing pharmaceutical manufacturing base (EIPICO, Amoun, Memphis) and has expanded post-2020 PPE production, the domestic medical device manufacturing sector remains narrow. Most diagnostic devices, specialty instruments, surgical sutures, and sterilization equipment still depend on a small number of authorized international distributors. Any disruption to a single distributor (e.g., exclusive-agreement termination or import licence suspension) can leave a clinic unable to source a critical consumable for weeks. Clinics should maintain at least two qualified vendors per critical category and pre-qualify alternatives before primary vendor contracts expire.
